# 横州防汛互助信息站

这是一个民间远程信息整理原型，用于展示已核验公开信息、官方渠道、物资需求、志愿协作入口和隐私保护规则。

## 项目边界

- 不代表官方机构。
- 不接收个人捐款，不放私人收款码，不代收物资。
- 不发布未经核验的求助信息。
- 不公开受灾群众手机号、门牌、身份证、精确定位等敏感信息。
- 不展示个人功劳或作秀内容。
- 所有真实救援、灾情发布、物资接收与捐赠安排均以官方渠道为准。

## 使用目的

真心希望能帮到别人，也真心不想给一线添乱。

这个页面适合用作：

- 公开渠道汇总页
- 已核验需求展示页
- 志愿者远程协作入口
- 信息核验和隐私保护规则示例

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## GitHub Pages 部署

仓库推送到 GitHub 后，在仓库设置中启用 GitHub Pages：

1. 进入 `Settings`。
2. 打开 `Pages`。
3. `Build and deployment` 选择 `GitHub Actions`。
4. 推送到 `main` 分支后会自动发布。

## 非商业说明

本项目仅建议用于公益、救灾互助、志愿协作、教育演示和非商业用途。请勿用于商业引流、广告推广、未经授权募捐、贩卖流量或制造灾情焦虑。

公开仓库不等于官方背书。使用者应自行核验信息来源，并遵守当地法律法规和现场应急指挥要求。
